# Інформаційна система управління бібліотекою

## Опис проєкту
Цей проєкт є модульною об'єктно-орієнтованою системою автоматизації процесів сучасної бібліотеки. Програма дозволяє вести облік книг, координувати права доступу користувачів, здійснювати гнучкий пошук у каталозі, а також миттєво сповіщати підписників про оновлення фонду.

Архітектура побудована на базі принципів **SOLID** та паттернів проєктування **GoF**, розроблених у ході виконання лабораторних робіт №1-6 з дисципліни "Конструювання ПЗ".

## Функціональність
- **Управління користувачами та ролями:** Поділ на базових користувачів, читачів (`Reader`) з контактними даними та бібліотекарів (`Librarian`) з рівнями доступу.
- **Каталог та пошук:** Гнучкий пошук книг за назвою, автором чи ID через абстрактний інтерфейс `BookSearch` (Dependency Injection).
- **Патерни проєктування:**
  - **Singleton:** Гарантує унікальність екземплярів класів `Catalog` та `Library` у пам'яті.
  - **Observer (Спостерігач):** Автоматична розсилка подій читачам та надсилання Email-адміністратору (`EmailAlert`) у разі додавання нових книг.
